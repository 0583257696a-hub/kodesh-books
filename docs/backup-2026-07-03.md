## קובץ הגיבוי

הגיבוי נוצר כארכיון ZIP נקי של הקוד מהקומיט היציב:

```text
backups/kodesh-books-stable-2026-07-03.zip
```

הגיבוי נוצר מתוך Git ולכן הוא כולל קבצי קוד ותצורה שמנוהלים בריפו בלבד.

## מה לא כלול בגיבוי

הגיבוי לא כולל:

- `.git`
- `node_modules`
- `dist`
- `.wrangler`
- קבצי build זמניים
- סודות, API keys או טוקנים
- שינויים לא מקומטים שהיו קיימים בזמן יצירת הגיבוי

## שחזור מהגיבוי

1. חלץ את קובץ ה-ZIP לתיקייה חדשה.
2. התקן תלויות:

```bash
npm install
```

3. בדוק build:

```bash
npm run build
```

4. אם צריך לפרוס ל-Cloudflare:

```bash
npm run deploy
```

## תלויות Cloudflare נדרשות

ב-Cloudflare חייבים להיות מוגדרים:

- D1 binding בשם `DB`
- R2 binding בשם `PRODUCT_IMAGES`
- Email binding בשם `EMAIL`
- `EMAIL_PROVIDER`
- `EMAIL_FROM`
- משתני Tranzila לפי המסוף בפועל

## הערה חשובה

קובץ הגיבוי הוא snapshot של מצב יציב. אין להכניס לתוכו סודות ידנית. סודות נשארים ב-Cloudflare Dashboard / Wrangler secrets בלבד.
